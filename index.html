DevServer

Setup interativo (N5105)

Bem-vindo ao seu Mini Servidor

Hardware: Celeron N5105 | 8GB RAM DDR4 | SSD NVMe

O Objetivo

Criar um ambiente de desenvolvimento limpo, performático e auto-hospedado para programar em Python, HTML (PWAs) e PowerShell, auxiliado por IA local.

A Arquitetura

Ubuntu Server rodando serviços nativos essenciais (IA, PowerShell) e isolando serviços secundários em contêineres Docker (Bancos de dados, Apps).

Navegue pelo menu lateral ou superior para ver as instruções. Todos os blocos de código possuem um botão de cópia no canto superior direito. Dica: Este PWA pode ser instalado no seu dispositivo.

1. Preparando o Ubuntu Server

Atualizar Pacotes e Repositórios

sudo apt update && sudo apt upgrade -y

Instalar Utilitários Básicos

sudo apt install curl wget git nano build-essential -y

2. Docker e PowerShell (Nativos)

Instalar o Docker via Script Oficial

curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

Adicione seu usuário ao grupo do Docker para evitar o 'sudo':

sudo usermod -aG docker $USER

Instalar o PowerShell Core

wget -q "https://packages.microsoft.com/config/ubuntu/$(lsb_release -rs)/packages-microsoft-prod.deb"
sudo dpkg -i packages-microsoft-prod.deb
sudo apt update
sudo apt install -y powershell

3. Instalando a IA Local (Ollama)

1

Instalar o Motor Ollama

curl -fsSL https://ollama.com/install.sh | sh
2

Liberar Acesso Remoto

sudo systemctl edit ollama.service

Cole as linhas abaixo:

[Service]
Environment="OLLAMA_HOST=0.0.0.0"
3

Baixar os Modelos Otimizados

ollama run qwen2.5-coder:1.5b
ollama run qwen2.5-coder:3b

4. Banco de Dados Leve (PocketBase)

1. Crie a pasta e o arquivo de configuração:

mkdir -p ~/docker/pocketbase/pb_data
cd ~/docker/pocketbase
nano docker-compose.yml

2. Cole a configuração YAML no arquivo:

services:
  pocketbase:
    image: ghcr.io/muchobien/pocketbase:latest
    container_name: pocketbase
    restart: unless-stopped
    command:
      - --encryptionEnv
      - PB_ENCRYPTION_KEY
    environment:
      PB_ENCRYPTION_KEY: "uma_senha_forte_aqui"
    ports:
      - "8090:8090"
    volumes:
      - ./pb_data:/pb/pb_data

3. Suba o serviço:

docker compose up -d

5. Acesso e Integração

Encontrar o IP do Servidor

ip a | grep inet

Configurando o VS Code

"models": [
  {
    "title": "Qwen 2.5 (1.5b)",
    "provider": "ollama",
    "model": "qwen2.5-coder:1.5b",
    "apiBase": "http://IP_SERVIDOR:11434"
  }
]
