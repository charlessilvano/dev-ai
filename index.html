DevServer

Setup interativo (N5105)

Bem-vindo ao seu Mini Servidor

Hardware: Celeron N5105 | 8GB RAM DDR4 | SSD NVMe

O Objetivo

Criar um ambiente de desenvolvimento limpo, performático e auto-hospedado para programar em Python, HTML (PWAs) e PowerShell, auxiliado por IA local.

A Arquitetura

Ubuntu Server rodando serviços nativos essenciais (IA, PowerShell) e isolando serviços secundários em contêineres Docker (Bancos de dados, Apps).

Navegue pelo menu lateral ou superior para ver as instruções. Todos os blocos de código possuem um botão de cópia no canto superior direito. Dica: Este PWA pode ser instalado no seu dispositivo.

1. Preparando o Ubuntu Server

O primeiro passo após instalar o Ubuntu Server limpo é garantir que o sistema está atualizado e possui as ferramentas básicas para compilação e download.

Atualizar Pacotes e Repositórios

sudo apt update && sudo apt upgrade -y

Instalar Utilitários Básicos

Ferramentas como curl, wget e git são essenciais para os próximos passos.

sudo apt install curl wget git nano build-essential -y

2. Docker e PowerShell (Nativos)

Instalando os motores fundamentais de infraestrutura e automação diretamente no sistema operacional.

Instalar o Docker via Script Oficial

O método mais limpo para garantir a versão atualizada do Docker.

curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

Adicione seu usuário ao grupo do Docker para evitar o 'sudo':

sudo usermod -aG docker $USER
# Obs: É necessário sair (exit) e entrar novamente (SSH) para aplicar.

Instalar o PowerShell Core

A instalação nativa via repositório da Microsoft garante integração perfeita no Ubuntu.

# Baixa as chaves do repositório da Microsoft
wget -q "https://packages.microsoft.com/config/ubuntu/$(lsb_release -rs)/packages-microsoft-prod.deb"
sudo dpkg -i packages-microsoft-prod.deb
sudo apt update

# Instala o PowerShell
sudo apt install -y powershell

Para testar, digite pwsh no terminal.

3. Instalando a IA Local (Ollama)

Configuração do motor de inferência que irá rodar direto na CPU. A instalação será nativa para máxima performance.

1

Instalar o Motor Ollama

curl -fsSL https://ollama.com/install.sh | sh
2

Liberar Acesso Remoto (Crucial)

Para que o seu VS Code (no seu PC) se conecte ao Ollama, precisamos liberar a porta na rede.

sudo systemctl edit ollama.service

No editor que abrir, cole as linhas abaixo:

[Service]
Environment="OLLAMA_HOST=0.0.0.0"

Salve (Ctrl+O, Enter, Ctrl+X) e reinicie o serviço:

sudo systemctl daemon-reload
sudo systemctl restart ollama
3

Baixar os Modelos Otimizados

Execute estes comandos para baixar os modelos adequados para os 8GB de RAM.

ollama run qwen2.5-coder:1.5b
# Após baixar, digite /bye

ollama run qwen2.5-coder:3b
# Digite /bye para sair após o download

4. Banco de Dados Leve (PocketBase)

Usando Docker Compose para subir o banco de dados que consumirá apenas ~40MB de RAM.

Criar o Contêiner Docker

1. Crie a pasta e o arquivo de configuração:

mkdir -p ~/docker/pocketbase/pb_data
cd ~/docker/pocketbase
nano docker-compose.yml

2. Cole a configuração YAML no arquivo:

services:
  pocketbase:
    image: ghcr.io/muchobien/pocketbase:latest
    container_name: pocketbase
    restart: unless-stopped
    command:
      - --encryptionEnv
      - PB_ENCRYPTION_KEY
    environment:
      PB_ENCRYPTION_KEY: "uma_senha_forte_aqui"
    ports:
      - "8090:8090"
    volumes:
      - ./pb_data:/pb/pb_data

3. Suba o serviço em segundo plano:

docker compose up -d

5. Acesso e Integração

Como juntar tudo e conectar o seu computador principal aos serviços do Mini Servidor.

Encontrar o IP do Servidor

No terminal do servidor Ubuntu:

ip a | grep inet

Painel do PocketBase

No navegador do seu PC principal:

http://[IP_SERVIDOR]:8090/_/

Configurando o VS Code

  1. No seu PC principal, instale a extensão Continue.dev.
  2. Abra o arquivo config.json da extensão.
  3. Adicione a configuração:
"models": [
  {
    "title": "Qwen 2.5 (1.5b)",
    "provider": "ollama",
    "model": "qwen2.5-coder:1.5b",
    "apiBase": "http://IP_SERVIDOR:11434"
  }
]

Tudo Pronto!

O seu Celeron N5105 atua no fundo da rede processando a IA e os bancos de dados localmente e com eficiência.
